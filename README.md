# Food laboratory

This application is a small project that allows me to get back into using React Native. The main goal is to use its features and structure a project, rather than creating a great product.

The project was built from the official boilerplate then hand-coded. 

## The project

Food Laboratory is an application for tracking your food experiments over time. It's not about cooking, but about fermenting, brewing, curing — any project where you need to monitor changes and keep a record of your observations.  

## Run the project

Execute `npm start` then run the application through the application [Expo Go](https://expo.dev/go) or on the web.
